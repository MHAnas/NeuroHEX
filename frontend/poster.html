03 · Analysis Pipeline
From raw MRI to clinical report in one click.
  • 01
    Upload — WebGL MRI Rendering
    Drag .nii, .nii.gz, or .dcm files. NiiVue renders multiplanar views (axial, sagittal, coronal) + full 3D volume using WebGL shaders at 60 fps. A 256³ voxel brain (16M data points) rendered entirely on device.
    NiiVue WebGLdcmjs@0.52NIfTI/DICOM
  • 02
    Segment — SynthSeg
    Billot et al. (Nature Methods 2023) CNN performs true 32-region FreeSurfer parcellation in 45–90s on CPU — vs 8–12h for full FreeSurfer. Hippocampal (labels 17/53), thalamic, caudate, putamen volumes in mm³. Stateless FastAPI backend discards files immediately.
    SynthSeg CNN32 RegionsFreeSurfer Labels
  • 03
    Screen — BraTS
    BraTS 2020 ONNX model via FastAPI /tumor endpoint. Outputs peritumoral edema (label 2), enhancing tumor (label 4), necrotic core (label 1) in cm³. Volume >0.5cm³ triggers ICD-10 alert. Fazekas grading (0–3) for white-matter hyperintensities — critical for BD's hypertension-driven WMH burden.
    BraTS 2020ONNX RuntimeFazekas Criteria
  • 04
    Analyze — Decision Support System
    Hippocampal volumes benchmarked vs Raz et al. 2005 age/sex norms. Brain age via Cole et al. 2018 ridge regression — gap >10yr flags accelerated neurodegeneration. ICD-10 codes (G30, G35, I69.3, G20) auto-flagged. BD-validated: MTA score r=−0.350, p=0.005 at NINS&H [PMC11363468].
    Raz 2005 NormsCole 2018ICD-10PMC11363468
  • 05
    Report — Gemini AI Generation
    Gemini 2.0 Flash generates 5-section formal neuroradiology report (TECHNIQUE · FINDINGS · IMPRESSION · DIFFERENTIAL · RECOMMENDATIONS) in 2–3 seconds. Groq LLaMA 3.3 70B as fallback. Bengali interpretation for frontline clinicians. Export: HTML · JSON · PDF.
    Gemini 2.0 FlashGroq FallbackBengali output

05 · Design Principles
Six principles that governed every decision.
  • P1
    Zero Data Egress
    NIfTI parsing, DICOM reading, volumetric analysis, 3D rendering — all in-browser. Zero patient bytes transmitted.
  • P2
    Low-Time & Low-Cost Infrastructure
    SynthSeg on CPU, not GPU. No expensive cloud compute. No vendor license. Full analysis in <60 seconds.
  • P3
    Graceful Degradation — Never Fail Silently
    Backend offline → auto fallback to intensity-based segmentation with visible DEMO MODE warning.
  • P4
    Evidence-Based Scientific Foundations & Citation Metrics
    Every threshold, norm, and model cites peer-reviewed literature. Every footer carries citations.
  • P5
    Multimodal Dual-Language Interpretation
    English neuroradiology report + Bengali plain-language summary for frontline district GPs.
  • P6
    Advanced Visualization, Interactivity & Pre-surgery-planning
    3D surgical pin planner, 9-axis longitudinal radar chart, disease atlas with 5 conditions — clinical utility beyond reporting.
